## Install scripts and dotfiles

Scripts and dotfiles for a fresh install of Ubuntu. Entrypoint is ./fresh_install.sh. Tested on Ubuntu 20.04.1 LTS.

### Applications
- Anki
- Atom
- Calibre
- Docker
- Etcher
- Google Chrome
- Google Cloud Platform SDK
- Libreoffice
- Slack
- Steam
- Terminator
- Transmission
- VLC

### Languages
- Golang
- Node
- Python
- Ruby
